# AlgoForge Visualizer

AlgoForge Visualizer is an interactive Data Structures and Algorithms (DSA) visualization platform built using Next.js, TypeScript, and custom-built data structure implementations.

The project helps users understand how popular data structures work internally through real-time visual animations and operations.

## 📌 Features

### ✅ Stack Visualizer

* Push operation
* Pop operation
* LIFO (Last In First Out) visualization
* Animated stack rendering

### ✅ Queue Visualizer

* Enqueue operation
* Dequeue operation
* FIFO (First In First Out) visualization
* Dynamic queue updates

### ✅ Linked List Visualizer

* Append nodes
* Remove nodes
* Connected node visualization
* Pointer-style rendering

### ✅ Binary Search Tree Visualizer

* Dynamic BST insertion
* Recursive tree rendering
* Parent-child node structure
* Live tree visualization

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/nutanshinde1/algoforge-visualizer.git
```

Move into project folder:

```bash
cd algoforge-visualizer
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
