# AlgoForge Visualizer

Interactive DSA visualization platform built using Next.js, TypeScript and custom data structure implementations.

## Features

- Stack Visualizer
- Queue Visualizer
- Linked List Visualizer
- Binary Search Tree Visualizer

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
